### Setup & Installation

To get the project up running 

```
$ git clone https://github.com/Ritik-Sharma38/Starwisp.git
$ cd Starwisp
```

- if you use NPM
```
$ npm install 
$ npm start
```

- if you use yarn
```
$ yarn
$ yarn start
```

Open the [localhost](http://localhost:3000) with your browser to see the result.
